Project Dependencies

1. nodejs > 8.0
2. create-react-app

Project Setup

```
1. Download and extract the zip file 
2. cd sravani1906.github.io-release (Or Zip file name)
3. npm install
```

Running Project in local server

```
1. npm start
2. App is accessible in http://localhost:3000/#
```
